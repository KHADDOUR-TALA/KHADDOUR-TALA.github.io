Research Projects & Innovation

Hand Exoskeleton for Rehabilitation

Rehabilitation Robotics • Human-Machine Interface • Biomedical Engineering

A lightweight, 3D-printed hand exoskeleton designed for post-stroke rehabilitation. The system uses soft actuators and EMG signal processing to provide adaptive assistance for grasping and fine motor skill recovery.

  • Adaptive control based on patient's EMG signals
  • 3D-printed modular design for cost-effectiveness
  • Real-time progress tracking via mobile application
  • Weight: 280g | Actuation: Pneumatic soft actuators

SLAM • Computer Vision • Path Planning

EMG-based Prosthetic Control System

Biomedical Signal Processing • Machine Learning • Embedded Systems

A low-cost prosthetic control system using surface EMG signals with machine learning classification for intuitive hand gesture recognition. Designed for upper-limb amputees in resource-limited settings.

  • 95% gesture recognition accuracy with CNN model
  • Latency: < 150ms from signal to actuation
  • Custom PCB design for signal acquisition
  • Cost: < $500 (commercial systems: $5,000+)
