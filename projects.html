Featured Project

Hand Exoskeleton for Rehabilitation

Advanced robotic system for post-stroke hand rehabilitation

Project Overview

Development of a soft robotic exoskeleton system for hand rehabilitation, focusing on post-stroke patients with hand impairment. The system integrates advanced control algorithms with ergonomic design to provide adaptive assistance.

Research Motivation

Stroke affects millions worldwide, often resulting in hand mobility issues. Traditional therapy is labor-intensive and limited by therapist availability. This project aims to create an accessible, personalized rehabilitation solution that can be used in clinical and home settings.

Technical Contributions

Dynamic Modeling

Developed accurate biomechanical models of hand kinematics for precise control

Fuzzy Logic Control System

MATLAB, Simulink, Fuzzy Logic Toolbox

Comparative study of fuzzy logic controllers (FLC) versus traditional PID controllers for nonlinear systems. Designed and implemented various FLC architectures including P-like, PI-like, PD-like, and PID-like controllers.
